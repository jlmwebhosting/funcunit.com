siblings FuncUnit.prototype.siblings

 

siblings()

Asynchronous version of siblings. Performs the exact same functionality as the jQuery method but adds itself to the queue.

 // after the click, filter the collection, then wait for result to be visible
F(".foo").click().siblings().visible();
