dblclick

 

Dblclicks an element. This runs two [Syn.prototype.click click] events followed by a dblclick on the element.

Example

syndblclick(options, element, callback)

Dblclicks an element. This runs two [Syn.prototype.click click] events followed by a dblclick on the element.

Example

Parameters

  1. options {Object}

  2. element {HTMLElement}

  3. callback {function()}

Syn.dblclick({},'open')
